class eitprocessing.eit_data.eit_data_variant.EITDataVariant[source]¶

Bases: eitprocessing.variants.Variant, eitprocessing.mixins.slicing.SelectByTime

Contains a single variant of a dataset.

A variant of a dataset is defined as either the raw data, or an edited version of that raw data. For example, EIT data can contain a “raw” versions and a “filtered” version. Both variants share the same time axis.

The actual data of a variant is contained in a variable that must be set by a subclass inheriting from this class.

Attributes: - label (str): a short descriptor for the variant, that is used to access

the variant

  • description (str): a longer description of the variant

  • params (dict): contains information on how to reproduce the variant, e.g. which filters and filters settigs were used

Concatenates two variants.

Concatenating two variants results in a single variant with the combined length of both variants.

To merge more than two variants, use functools.reduce(Variant.concatenate, list_of_variants).

Args: - a (Variant) - b (Variant)

Raises: - EquivalenceError if a and b are not equivalent and can’t be merged
